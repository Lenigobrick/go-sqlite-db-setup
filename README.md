# go-sqlite-db-setup
sqlite DB template for go using database/sql and modernc.org/sqlite as driver

### Prerequisites
1. Setup : (delete the go.mod and go.sum before if you do that)
```sh
go mod init GoSqliteDbSetup 
```
```sh
go get modernc.org/sqlite@latest
```

2. If you want to update/upgrade : 
```sh
go mod tidy
```

## Usage

3. Run :
```sh
go run .
```
